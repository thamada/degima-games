# 設計仕様書

> **注意**: 本ドキュメントは設計仕様書です。変更履歴や実装の詳細な変更点については、`ChangeLog.md`を参照してください。本ドキュメントでは、現在のシステムの設計と仕様を記述します。

## 概要

degima-games はブラウザで遊べるレトロゲーム集です。単一ページ内のタブで「インベーダー」「ポン」「メビウス」を切り替えてプレイできます。ゲーム本体は HTML5 Canvas 上のスタンドアロン HTML として実装し、シェルから iframe 経由で読み込みます。

## ディレクトリ構成

## アーキテクチャ

### ゲームシェル（games/index.html）

- 高さ 32px のコンパクトなタブバーを画面上部に配置
- 各ゲームは iframe で読み込み（ゲーム間でスクリプト・状態を分離）
- 起動時は「メビウス」を表示。「インベーダー」「ポン」は初回タブ選択時に遅延読み込み（`data-src`）
- シェル全体および iframe 領域は `overflow: hidden` とし、ページスクロールを発生させない

### 各ゲーム HTML

- Canvas 解像度: 640×480（内部描画バッファ）
- 表示サイズは CSS の `max-width` / `max-height` でウィンドウ内に収まるよう自動縮小
- レトロ風 UI（黒背景・緑のネオン風ボーダー）
- ゲーム操作キー（矢印キー・スペース等）では `preventDefault()` を呼び、ブラウザのスクロール動作を抑止

## ゲーム仕様

### インベーダー（invaders.html）

| 項目 | 内容 |
|------|------|
| 操作 | ←→ 移動、スペース 発射、R リスタート（ゲームオーバー時） |
| ルール | インベーダーを撃破してスコアを稼ぐ。被弾で残機減少、全滅または侵入でゲームオーバー |
| 残機 | 初期 3。敵弾被弾のたびに 1 減少 |
| ウェーブ | クリアごとに敵数・移動速度が増加 |

#### HUD

- 左上: `SCORE`（スコア）と `WAVE`（ウェーブ数）をテキスト表示
- 右上: `LIVES` ラベルと、残機数分の小型自機アイコンを横並び表示
- 被弾時は爆発開始と同時に残機アイコンが 1 つ減る

#### 自機被弾・爆発

- 敵弾命中時: 残機を 1 減らし、自機位置で爆発エフェクトを再生（自機は非表示）
- 爆発中: ゲーム進行を一時停止（約 50 フレーム）
- 爆発演出: 中心フラッシュ、拡大リング、黄〜赤〜緑のパーティクル飛散
- 爆発終了後: 残機が残っていれば画面中央下に自機を復活。残機 0 ならゲームオーバー

#### 敵の攻撃

- 射手: 各列の最前列（プレイヤーに最も近い敵）のみが射撃可能
- 射撃確率（フレームごと）: `0.018 + wave × 0.005`。残存敵が 20 体以下で ×1.4、10 体以下でさらに ×1.3
- 弾速: `5 + wave × 0.6`
- 同時射撃: ウェーブ 2 以降、35% の確率で別列から 2 発目を追加発射
- 敵弾は赤色（`#f00`）で描画。プレイヤー被弾で残機 1 減少し爆発演出が発生

### ポン（pong.html）

| 項目 | 内容 |
|------|------|
| 操作 | ↑↓ 移動（左パドル）、R リスタート |
| ルール | 右側は CPU パドル。相手側ゴールで得点 |
| 備考 | ボール速度はラケット反射のたびにわずかに増加 |

#### 自機ミス・爆発・溶解

- 左パドルでボールを逃したとき（`ball.x < 0`）: CPU 側（`scoreR`）に 1 点加算
- ボール爆発: ボールを非表示にし、命中位置で緑系パーティクルを放射状に飛散（重力付き）
- 自機溶解: 左パドルを非表示にし、パドル領域からパーティクルを生成して拡散・フェードアウト
- 演出中: ゲーム状態を `missAnim` に切り替え、操作・ボール移動を一時停止（約 90 フレーム）
- 演出終了後: ボールを中央へリセットし、左パドルを再表示して `playing` に復帰
- CPU 側得点時（`ball.x > canvas.width`）: 爆発・溶解なしで即座にボールのみリセット

### メビウス（mevius.html）

縦スクロールシューティング。地上が上から下へ流れ、空中敵と地上目標を同時に攻略する。

| 項目 | 内容 |
|------|------|
| 操作 | ←→↑↓ 移動、スペース 対空（ZAPPER）、X 対地（BLASTER）、R リスタート（ゲームオーバー時） |
| ルール | スクロール地形上の敵を撃破してスコアを稼ぐ。被弾で残機減少、残機 0 でゲームオーバー |
| 残機 | 初期 4 |
| ウェーブ | スコア `wave × 2000` 到達ごとに上昇。敵出現・スクロール速度が増加 |

#### 武器（基本）

| 武器 | キー | 対象 | 備考 |
|------|------|------|------|
| ZAPPER | スペース | 空中敵 | 直進弾（シアン）。クールダウン 5 フレーム |
| BLASTER | X | 地上目標（空中敵にも有効） | 斜め落下弾（黄色）。クールダウン 12 フレーム |

#### アイテム（パワーアップ）

敵（空中・地上）撃破時に **約 17%** の確率でドロップ。自機接触で取得。

| アイテム | 表示 | 効果 | 持続 |
|----------|------|------|------|
| R1〜R4 | 緑〜橙・放射線 | 対空 RAD ビーム Lv1〜4（Lv ごとに弾数・威力・連射速度が増加。Lv4 は暖色細線レーザーの連射） | 540 フレーム（約 9 秒） |
| FLM | 橙・炎 | 対地 BLASTER 強化（3 方向火炎弾・連射速アップ） | 540 フレーム |
| F1〜F4 | 桃〜金・ミサイル | ファンネルミサイル（誘導弾）Lv1〜4 を自動発射 | 540 フレーム |
| B1 | 水色リング 1 重 | バリア +1（最大 8 段。Ξ モード中は最大 16 段） | 時間制限なし |
| B2 | 水色リング 2 重 | バリア +2（最大 8 段。Ξ モード中は最大 16 段） | 時間制限なし |
| B8 | 金色リング 8 重 | バリアを 8 段に設定（Ξ モード中は 16 段） | 時間制限なし |
| STR | 金色五芒星 | 無敵（敵弾ダメージ無効） | 480 フレーム（約 8 秒） |
| WFR | 橙・爆風＋炎 | 画面内の地上敵を一括撃破し、約 3 秒間の山火事状態を発生 | 180 フレーム（約 3 秒） |
| Ξ | 白・V字シルエット | 自機を Ξ ガンダム風ロボに変形。バリア 16 段＋高速レーザービームのみ装備 | 900 フレーム（約 15 秒） |

##### 山火事（WFR）

取得時に画面内の地上目標を即座に撃破（スコア加算）。画面下半分の地上砲弾も消去する。約 3 秒間、背景が焦げ色に変化し爆風炎の演出が表示される。山火事中は画面下半分の地上目標に継続ダメージ（約 0.3 秒ごとに HP-2）が入り、新規地上目標の出現率が低下する。空中敵・自機には影響しない。ドロップ率は約 17% 中 2%（レア）。

演出内容:

- 取得直後: 画面フラッシュ、カメラシェイク、地上複数箇所からの爆風炎（白熱コア・橙の炎柱・黒煙）
- 継続中: 衝撃波リング、炎・煙・火花パーティクル、大気の橙霞、自機への逆光リムライト
- 描画レイヤー: 背景焦げ色 → 爆風バックドロップ → ゲームオブジェクト → 爆風フロント（明るい炎核・パーティクル）

デバッグ用出現: ゲーム開始・R リスタート直後に自機付近へ WFR を 1 個出現。さらに **10 秒（600 フレーム）後** に WFR を再出現。**15 秒（900 フレーム）後** に Ξ アイテムを自機付近へ出現。

##### 開始時装備（STARTING_LOADOUT）

`STARTING_LOADOUT` 定数でゲーム開始時および R キーリスタート時の初期装備を設定可能。各項目を `true` にすると、対応するパワーアップ効果を付与した状態で開始する（デフォルト: **RAD Lv3** / FLM / B8 / STR をすべて有効）。被弾・撃墜時は従来どおり武器強化・バリア・無敵はリセットされる。

##### RAD ビーム（R1〜R4）

取得中、スペースキーで対空 ZAPPER が RAD ビームに置き換わる。既存 Lv より高い R アイテム取得時は Lv アップ（低 Lv 取得時は Lv 維持・タイマーのみ延長）。

| Lv | 弾形 | 同時発射 | 代表ダメージ | 連射間隔（フレーム） |
|----|------|----------|-------------|---------------------|
| 1 | 中央 1 本・貫通 | 1 | 2 | 5 |
| 2 | 中央＋左右・貫通 | 3 | 2 / 1 | 4 |
| 3 | 中央＋左右＋外側・貫通 | 4 | 3 / 2 / 1 | 3 |
| 4 | 暖色細線レーザー（1 本ずつ連射） | 1（命中後に次を発射） | 4 | 自動連射 |

- **Lv1〜3**: 従来型の緑色貫通ビーム（太さ・本数・速度が Lv に応じて増加）。スペースキーで発射
- **Lv4**: Ξ モードの高速レーザービームと同じ細線描画（暖色系オレンジ）。最寄りの空中敵へ照準し、1 本ずつ飛翔・命中後に次弾を連射。武器タイマー中は自動連射（スペース不要）

##### Ξ（クスィー）アイテム

取得時に戦闘機（32×36px）から Ξ ガンダム風ロボ（48×52px）へ変形。約 15 秒間、以下の状態になる。

| 項目 | 内容 |
|------|------|
| 武器 | **高速レーザービームのみ**（ファネルミサイル・フレイム・RAD ビーム等は使用不可） |
| レーザー | 緑色の細線。1 本ずつ最寄りの空中・地上敵へ照準し、命中後に次弾を連射 |
| バリア | 16 段（六角シールド演出）。被弾時 1 段消費 |
| 終了 | タイマー満了または撃墜で戦闘機に復帰。残バリアは最大 8 段まで繰越 |

ドロップ率は敵撃破時 17% 中 1.5%（レア）。

##### ファンネルミサイル（F1〜F4）

取得中、一定間隔で自機左右から誘導ミサイルを自動発射。最も近い空中敵または地上目標を追尾し、ターゲット消失後は再ロックオンする。既存 Lv より高い F アイテム取得時は Lv アップ（低 Lv 取得時は Lv 維持・タイマーのみ延長）。

| Lv | 同時発射数 | 弾速（基準比） | ダメージ | 発射間隔（フレーム） |
|----|-----------|---------------|----------|---------------------|
| 1 | 2 | ×1（6） | 2 | 22 |
| 2 | 4 | ×2（12） | 2 | 22 |
| 3 | 6 | ×3（18） | 2 | 20 |
| 4 | 8 | ×4（24） | 3 | 18 |

##### バリア・無敵

- **バリア**: 敵弾命中時に 1 段消費して被弾を防ぐ（残機は減らない）。自機撃墜時にリセット
- **無敵スター**: 無敵中は敵弾によるダメージ・バリア消費ともに発生しない。自機は点滅＋星オーラ表示。撃墜時に解除
- 被弾判定の優先: 無敵 → バリア → 残機減少

##### 武器強化のリセット

自機撃墜時、RAD / FLM / ファンネルミサイルの強化状態および Ξ モードはすべて解除される。

#### 空中敵

- 種類: 小型 / 大型（HP 1 / 2）
- 移動パターン: 直進（straight）、蛇行（sine）、急降下（dive）
- シアン〜赤系の発光シルエット、`AIR` ラベル、浮遊影で空中であることを明示
- 弾速: `3.5 + wave × 0.2`（小型赤弾）

#### 地上目標

- 種類: 戦車（tank）、対空砲（aaGun）、トーチカ（pillbox）、小型要塞（smallFortress）、巨大要塞（giantFortress）
- 茶〜黄系の接地色、`GROUND` ラベル、黄色のターゲットリングで地上であることを明示
- 対空砲・巨大要塞は自機照準の砲弾を発射
- 地上砲弾速度: 対空砲 `1.35`、巨大要塞 `1.15`（銅色の砲弾シルエット＋発射煙トレイル。進行方向に先端が向く）

#### 自機

- **通常時**: F-16 風のトップビューシルエット（32×36px）。グレー系塗装・デルタ翼・コックピットキャノピー
- **Ξ モード時**: オフホワイト装甲・巨大ショルダーバインダー・V字アンテナ・青白スラスターの Ξ ガンダム風シルエット（48×52px）
- ←→ 移動時: 左右バンク（最大約 0.38 rad）を滑らかに補間
- ↑ 前進時: エンジン後方からアフターバーナー（炎の揺らめき＋火花）を再生。キー離しでフェードアウト
- 地上影はバンクに追従せず、機体のみ回転

#### 背景

- 草原・森・畑パッチ・道路のスクロール地形（上から下へ流れる）
- ナスカの地上絵（コンドル、クモ、サル、渦巻き、台形）を巨大な溝状ライン＋光輪で描画
- 湖は描画しない

#### HUD

- 左上: `SCORE` / `WAVE`
- 左上（下段）: 取得中パワーアップの状態（`RAD LvN` / `RAD LASER`（Lv4） / `FLAME` / `FUNNEL LvN` / `Ξ LASER` / `Ξ BARRIER` / `INV STAR` / `WILDFIRE` の残り時間バー、`BARRIER` の段数表示）
- 右上: `LIVES` と残機アイコン
- 左下: 操作・アイテム凡例（`AIR: SPACE` / `GROUND: X`、R1〜R4 / FLM / B1〜B8 / STR / WFR / F1〜F4 / Ξ）。ゲーム開始・リスポーン直後に表示し、10 秒（600 フレーム）かけてフェードアウト。被弾〜爆発中は非表示

#### 自機被弾・爆発

- 敵弾命中時: 無敵中は無視。バリアがあれば 1 段消費して弾のみ消去。いずれもなければ残機を 1 減らし爆発
- 爆発中: ゲーム進行を一時停止（約 45 フレーム）。武器強化・バリア・無敵・誘導弾はすべてリセット
- 爆発終了後: 残機が残っていれば画面中央下に復活。残機 0 ならゲームオーバー

## UI 仕様（タブ）

- 右寄せのテキストタブ 3 件（インベーダー / ポン / メビウス）
- 非選択: グレー文字（`#666`）
- 選択中: 明るいグリーン文字（`#6ee06e`）＋下線インジケーター（`#3cb83c`）
- フォント: 等幅系（SF Mono / Menlo / Consolas）

## 開発・実行

| コマンド | 説明 |
|----------|------|
| `make` | デフォルトターゲット（`run` と同じ）。依存関係インストール後、`npm start` で開発サーバー起動（http://localhost:3000） |
| `make run` | 上記と同じ |
| `make clean` | `.local`、`node_modules` を削除 |

Makefile は `.local` 配下に Node.js v22.14.0 を取得し、プロジェクトローカルで npm を実行します。`serve` が `games/` ディレクトリを静的配信します（`-n` / `--no-clipboard` により URL のクリップボードコピーを行わない。Docker 等のヘッドレス環境向け）。
